# WanAndroid-MVVM
玩 Android 客户端，采用 kotlin 语言，Material Design 风格，根据 MVVM 架构使用 Jetpack 架构组件搭建了整套框架

持续更新ing~

### Jetpack 使用情况
- DataBinding
- Navigation
- LiveData
- ViewModel
- Paging
- Room (以后可能会用到)
